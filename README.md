# History Visualizer (-10M Years to Present)

## Overview

An interactive web application providing a "movie-like" experience for exploring human history through time and space.

### Key Features (MVP)

- **Timeline Slider**: Control time from -10,000,000 BC to 2026 AD.
- **Paleomap Layers**: Shifting continents and ice sheets based on the selected era.
- **Human Migration**: Conceptual visualization of population movements and lineages.
- **Civilization Layers**: Emergence of cultures and cities.
- **Life Expectancy Graph**: Real-time stats synchronized with the timeline.

## Directory Structure

- `docs/`: Requirements and research notes.
- `src/`: Source code (React + TypeScript).
- `data/`: JSON/GeoJSON datasets for history and maps.
